# Portfolio Frontend

## Setup

```bash
npm install

# .env already hai, check karo:
# VITE_API_URL=http://localhost:5000/api

npm run dev
# Runs on http://localhost:3001
```

## Important
Pehle backend start karo (port 5000), phir frontend (port 3001).

## Pages
/ → Home (Hero + About + Projects)
/projects → All Projects
/projects/:slug → Project Detail
/about → About page
/contact → Contact form
